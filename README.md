# Number Guessing Game

This is a simple number guessing game implemented in Python. The program generates a random number between 1 and 100, and the player has to guess the number within the fewest attempts possible.

## How to Run
- Make sure you have Python installed on your system.
- Clone this repository.
- Navigate to the directory containing the project files in your terminal or command prompt.
- Run the command `python number_guessing_game.py`.
- Follow the instructions to play the game.

Have fun guessing!

# URL Shortener

This Python script allows you to shorten long URLs using a URL shortening service's API. It provides a convenient way to generate shorter, more manageable URLs for sharing or embedding.

## Features
- Shorten long URLs to make them more concise.
- Supports various URL shortening services.
- Simple command-line interface.

## How to Use
1. Clone this repository to your local machine.
2. Install the required dependencies by running `pip install -r requirements.txt`.
3. Run the script using the command `python url_shortener.py`.
4. Enter the long URL that you want to shorten.
5. The script will generate a shortened URL using the specified URL shortening service and display it to you.

Start shortening URLs now!
